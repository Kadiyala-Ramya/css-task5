Combinator Selectors

1)Descendant Selector

It applies styles to all descendants (children, grandchildren, great-grandchildren, etc.) of a specified element.

Example: 

            html
            < div >
            < p >This will be red< /p >
               < div >
                    < p >This will be red< /p >
               < /div >
            < /div >
            < p >This will not be red< /p >

            Css
            div p 
              {
                color: red;
              }

2)Child Selector

It only selects elements that are one level down in the hierarchy from the specified parent.

Example:

               html
               < ul >
                  < li >Direct child – styled green< /li >
                  < li >
                    < ul >
                        < li >Nested child – NOT styled< /li >
                    < /ul >
                  < /li >
              < /ul >

              Css
              ul > li
               {
                color: green;
               }

3) Adjacent Sibling Selector

Selects an element that is immediately next to a specified element (must share the same parent).

Example:

            Html
            < h2 >Heading< /h2 >
            < p >Styled paragraph< /p >
            < p >Not styled< /p >

            Css 
            h2 + p 
              {
                color: red;
              }

4)General Sibling Selector

Selects all siblings that come after a specified element (not just the next one).

Example:


            Html 
            < h2 >Heading< /h2 >
            < p >Styled paragraph< /p >
            < p >Also styled< /p >

            Css 
            h2 ~ p 
            {
                color: purple;
            }
